# MAJA Sportswear – Data Analytics Portfolio

Este portafolio presenta proyectos de análisis de datos basados en un caso
realista de retail deportivo para MAJA Sportswear (Culiacán, Sinaloa).

## Objetivo
Demostrar mi capacidad para:
- Analizar datos comerciales reales
- Traducir datos en decisiones de negocio
- Comunicar hallazgos a audiencias no técnicas

## Herramientas utilizadas
- Excel (análisis exploratorio)
- SQL (MySQL)
- Power BI (visualización ejecutiva)

## Proyectos
- Proyecto 1: Análisis de ventas por tienda y producto
